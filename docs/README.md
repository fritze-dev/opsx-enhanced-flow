# Documentation

The opsx-enhanced plugin uses a three-layer architecture (Constitution, Schema, Skills) to deliver a spec-driven development workflow where every feature produces research, specs, architecture, quality checks, changelogs, and user docs alongside code.

| Section | Description |
|---------|-------------|
| [Architecture](architecture.md) | System architecture, tech stack, and conventions |
| [Design Decisions](decisions.md) | Key design decisions with rationale and trade-offs |
